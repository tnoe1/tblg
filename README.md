# tblg

A basic blog platform.

## Overview

tblg is a personal blogging platform.

## Features

- Easy migrations
- Lightweight sqlite db using `better-sqlite3`

Will have
- Create and publish blog posts
- View a list of all posts
- Read individual posts
- Edit and delete existing posts

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)

### Installation

```bash
git clone https://github.com/tnoe1/tblg.git
cd tblg
npm install
```

### Running the App

```bash
node index.js
```
